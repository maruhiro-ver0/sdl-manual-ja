Mix_Linked_Version

const SDL_version *Mix_Linked_Version()
void SDL_MIXER_VERSION(SDL_version *compile_version)

これはSDL_GetVersionやSDL_VERSIONと同じような機能を持つ. これらを使うと, 実行時のバージョンとコンパイル時のバージョンを比較できる.

サンプルコード

SDL_version compile_version;
const SDL_version *link_version=Mix_Linked_Version();
SDL_MIXER_VERSION(&compile_version);
printf("コンパイル時のSDL_mixerのバージョン: %d.%d.%d\n", 
        compile_version.major,
        compile_version.minor,
        compile_version.patch);
printf("実行中のSDL_mixerのバージョン: %d.%d.%d\n", 
        link_version->major,
        link_version->minor,
        link_version->patch);
