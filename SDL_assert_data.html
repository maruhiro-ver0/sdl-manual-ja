SDL_assert_data

アサーションの情報の構造体

フィールド

intalways_ignore設定されていれば1, いなければ0
unsigned inttrigger_countアサーションの回数
const char*conditionチェックするアサーションの条件
const char*filenameアサーションのあるファイルの名前
intlinenumアサーションのある位置
const char*functionアサーションのある関数
const struct SDL_assert_data*next次のアサーション情報へのポインタ

詳細

この構造体はアサーションハンドラで情報が書き込まれ, 報告に加えられる. SDL_GetAssertionReport()はリンクされたリスト構造でこの構造体を戻す.
