IPaddress

typedef struct {
    Uint32 host;            /* 32bit IPv4ホストアドレス */
    Uint16 port;            /* 16bit プロトコルポート */
} IPaddress;

host
ネットワークバイト順でエンコードされたホストのIPV4アドレス
port
ネットワークバイト順でエンコードされたソケットのIPv4ポート番号

ネットワーク接続とソケットで使われる情報を持つ型

参照

名前解決関数, SDLNet_TCP_Open, UDPpacket
