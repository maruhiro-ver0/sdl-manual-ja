SDL_GetWindowSize

Use this function to get the size of a window's client area. ウィンドウのクライアント領域のサイズを得る

構文

void SDL_GetWindowSize(SDL_Window* window, int* w, int* h)

引数

window問い合わせるウィンドウ
xウィンドウの幅を代入するポインタ
yウィンドウの高さを代入するポインタ
