SDL_AndroidShowToast

Androidトースト通知を表示する

構文

int SDL_AndroidShowToast(const char* message, int duration, int gravity, int xoffset, int yoffset)

引数

message表示するテキストメッセージ
duration0=短い, 1=長い
gravity通知を表示する画面上の位置
xoffsetgravity>=0の場合のみこの引数が設定される
yoffsetgravity>=0の場合のみこの引数が設定される

戻り値

成功のとき0, 失敗のとき-1を戻す.

詳細

トーストはAndroid固有の軽い注意喚起の一種である.
https://developer.android.com/guide/topics/ui/notifiers/toasts

トーストはUIスレッドで表示される.

引数gravityは下の値から選ぶこと. 適当なものがない場合は-1とすること.
https://developer.android.com/reference/android/view/Gravity

バージョン

この関数はバージョン2.0.16以降で使える
