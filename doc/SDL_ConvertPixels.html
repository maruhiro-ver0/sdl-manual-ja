SDL_ConvertPixels

ある形式のピクセル群を別の形式に変換してコピーする

構文

int SDL_ConvertPixels(int width, int height, Uint32 src_format, const void* src, int src_pitch, Uint32 dst_format, void* dst, int dst_pitch)

引数

widthピクセル群の幅
heightピクセル群の高さ
src_formatコピー元のピクセル形式 (詳細を参照すること)
srcコピー元のピクセル群
src_pitchコピー元のピクセル群の水平方向のバイト数
dst_formatコピー先のピクセル形式 (詳細を参照すること)
dst生成したピクセル群を書き込むポインタ
dst_pitchコピー先のピクセル群の水平方向のバイト数

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

src_formatとdst_formatはSDL_PixelFormatEnumの値の1つである.

バージョン

SDL 2.0.0以降
