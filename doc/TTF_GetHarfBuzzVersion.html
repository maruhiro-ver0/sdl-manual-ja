TTF_GetHarfBuzzVersion

使用しているHarfBuzzライブラリのバージョンを得る

構文

void TTF_GetHarfBuzzVersion(int *major, int *minor, int *patch)

引数

majorメジャーバージョンの代入先へのポインタ. NULLも可
minorマイナーバージョンの代入先へのポインタ. NULLも可
patchパッチバージョンの代入先へのポインタ. NULLも可

詳細

HarfBuzzを利用できない場合はバージョン0.0.0が報告される.

バージョン

SDL_ttf 2.0.18以降
