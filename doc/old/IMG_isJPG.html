IMG_isJPG

int IMG_isJPG(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがJPG形式に対応しており, 画像データをJPGとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がJPGでSDL_ImageがJPG形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.jpgがJPGかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.jpg", "rb");
if(IMG_isJPG(rwop))
    printf("sample.jpgはJPGファイルである\n");
else
    printf("sample.jpgはJPGファイルでないか, JPGに対応していない\n");
