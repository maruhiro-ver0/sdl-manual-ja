SDLNet_Init

int SDLNet_Init()

ネットワークAPIを初期化する. このライブラリの他の関数を呼ぶ前に, この関数を呼ぶ必要がある. この関数を呼ぶ前にSDLが初期化されている必要がある.

戻り値: 成功のとき0, 失敗のとき-1

サンプルコード

if(SDL_Init(0)==-1) {
    printf("SDL_Init: %s\n", SDL_GetError());
    exit(1);
}
if(SDLNet_Init()==-1) {
    printf("SDLNet_Init: %s\n", SDLNet_GetError());
    exit(2);
}
