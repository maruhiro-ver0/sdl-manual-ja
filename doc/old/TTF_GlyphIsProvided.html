目次 - 3.関数 - 3.3属性 - TTF_GlyphIsProvided

TTF_GlyphIsProvided

int TTF_GlyphIsProvided(const TTF_Font *font, Uint16 ch)

font
グリフが使えるかテストするフォント
ch
グリフが使えるかテストするUNICODE文字

fontで指定のUNICODE文字chが使えるかテストする.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: fontのUNICODE文字chのグリフの番号. 定義されてない文字のとき0

サンプルコード

// 読み込まれたフォントのグリフ'g'をチェックする
//TTF_Font *font;

int index=TTF_GlyphIsProvided(font,'g');
if(!index)
    printf("フォントに'g'がない!\n");
