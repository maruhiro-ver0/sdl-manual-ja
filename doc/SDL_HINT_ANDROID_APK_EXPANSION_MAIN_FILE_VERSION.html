SDL_HINT_ANDROID_APK_EXPANSION_MAIN_FILE_VERSION

APK拡張ファイルのメインファイルのバージョンを指定するヒント

値

XAPK拡張ファイルのメインファイルのバージョン ("1"や"2"のような数字の文字列である必要がある)

デフォルト

デフォルトではこのヒントは設定されない.

詳細

このヒントはSDL_HINT_ANDROID_APK_EXPANSION_PATCH_FILE_VERSIONと共に使う必要がある.

これらのヒントが設定されれば, SDL_RWFromFile()は与えられた相対パスが内部ストレージになければ拡張ファイルを検索しそれを使う.

デフォルトではこのヒントは設定されておらず, APK拡張ファイルは検索されない.

バージョン

SDL 2.0.4以降
