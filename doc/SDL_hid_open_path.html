SDL_hid_open_path

HIDデバイスを名前で開く

構文

SDL_hid_device * SDL_hid_open_path(const char *path, int bExclusive /* = false */)

引数

pathオープンするデバイスのパス名

戻り値

SDL_hid_deviceオブジェクトへのポインタを戻す. 失敗のときNULLを戻す

詳細

パス名はSDL_hid_enumerate()を呼んで決める. 環境固有のパス名(例: Linuxならば /dev/hidraw0 等)も使える.

バージョン

SDL 2.0.18以降
