SDL_HapticStopEffect

ハプティックデバイスで再生中の効果を停止する

構文

int SDL_HapticStopEffect(SDL_Haptic* haptic, int effect)

引数

haptic再生を停止するハプティックデバイス
effect停止する効果のID

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
