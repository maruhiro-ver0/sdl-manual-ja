2.始めに

既にSDL_mixerを入手し, システムにインストールしていることを前提とする. SDL_mixerのソースコードアーカイブに含まれるREADME文書は, コンパイルとインストールの手助けになるだろう.

一般的なUNIX-like環境と同様に, 以下の組み合わせでインストールする:

./configure
make
make install

SDL_mixerは以下の形式の音楽とサンプリング音声に対応している:
- WAVE/RIFF (.wav)
- AIFF (.aiff)
- VOC (.voc)
- MOD (.mod .xm .s3m .669 .it .medなど) libmikmodが必要
- MIDI (.mid) timidityまたはMIDIハードウェアを使用
- OggVorbis (.ogg) ogg/vorbisライブラリが必要
- MP3 (.mp3) SMPEGまたはMADライブラリが必要
- FLAC (.flac) FLACライブラリとコマンドラインのプレイヤーが必要 SDL_mixerではミキシングできない...

サンプルコードが必要ならば, 次の場所でダウンロードできる:
http://www.jonatkins.org/SDL_mixer/
