SDL_SetError

SDLエラー文字列を設定する

構文

int SDL_SetError(const char* fmt, ...)

引数

fmtprintf()形式の文字列形式
...fmt文字列の%トークンに一致する追加の引数

戻り値

常に-1を戻す.

サンプルコード

int errorCode = 0;
...
errorCode = -37;
...
if (errorCode < 0)
    SDL_SetError("Something unexpected happened: Error Code %d", errorCode);

詳細

この関数を呼ぶと, 以前のエラーメッセージは上書きされる.
