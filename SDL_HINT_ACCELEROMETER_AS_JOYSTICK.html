SDL_HINT_ACCELEROMETER_AS_JOYSTICK

Android/iOSの加速度センサーをジョイスティックの一覧に加えるか否かのヒント

値

0本物のジョイスティックのみ
1本物に加え, 加速度センサーを3軸のジョイスティックとする (デフォルト)

デフォルト

デフォルトでは, 本物に加え, 加速度センサーを3軸のジョイスティックとして一覧にする.

サンプルコード

#include "SDL.h"

int main(int argc, char* argv[])
{
    // 加速度センサーをジョイスティックとして使用しない
    SDL_SetHint(SDL_HINT_ACCELEROMETER_AS_JOYSTICK, "0");
}

詳細

iOSの場合はこのヒントはSDL 2.0.4以降で使える.

バージョン

SDL 2.0.2以降
