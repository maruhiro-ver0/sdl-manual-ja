SDL_RenderSetLogicalSize

レンダリングのデバイスに依存しない分解能を設定する

構文

int SDL_RenderSetLogicalSize(SDL_Renderer* renderer, int w, int h)

引数

rendererレンダリングコンテキスト
w論理幅
h論理高さ

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
