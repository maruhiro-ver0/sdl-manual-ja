SDLNet_GenericSocket

typedef struct {
    int ready;
} *SDLNet_GenericSocket;

ready
0でないときデータを読む準備ができている, またはサーバソケットが接続をACCEPTする準備ができている

このデータ型はUDPsocketとTCPsocket型の両方で使える. SDLNet_CheckSocketsを呼んだ後, そのソケットにSDLNet_SocketReadyを使うと, readyがソケットの活動にあわせて設定される. この型はこの使い方のためにだけ存在する. そのため, ソケットの種類を知るためには使えない.
