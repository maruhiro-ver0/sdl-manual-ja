SDL_FillRects

指定の色で高速に塗りつぶした長方形を複数描く

構文

int SDL_FillRects(SDL_Surface* dst, const SDL_Rect* rects, int count, Uint32 color)

引数

dst描画するサーフェイスのSDL_Surface構造体
rect塗りつぶされる領域を表すSDL_Rect構造体の配列
count塗りつぶされる領域の配列の個数
color塗りつぶす色 (詳細を参照すること)

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. このときSDL_GetError()を呼んで詳細を知ることができる.

詳細

colorはサーフェイスのピクセル形式である必要がある. SDL_MapRGB()またはSDL_MapRGBA()で生成することができる. 色にアルファ値があっても, 単にそのアルファ値が書き込まれる. ブレンディングは行われない.

サーフェイスにクリッピング領域が設定されていれば(SDL_SetClipRect()で設定する), この関数はクリッピング領域とrectsにしたがって塗りつぶす.
