SDL_SetClipRect

Use this function to set the clipping rectangle for a surface. サーフェイスのクリッピング領域を設定する

構文

SDL_bool SDL_SetClipRect(SDL_Surface* surface, const SDL_Rect* rect)

引数

surfaceクリッピング領域を設定するサーフェイスのSDL_Surface構造体
rectクリッピングする領域を表すSDL_Rect構造体. NULLのときクリッピングを無効にする

戻り値

領域がサーフェイスの内側にあるときSDL_TRUE, 転送が完全にクリッピングされてしまうときSDL_FALSEを戻す.

詳細

surfaceが転送先のとき, クリッピング領域の内側のみに描画される.

転送元と転送先の端で自動的にクリッピングされることに注意すること.
