SDLNet_UDP_Close

void SDLNet_UDP_Close(UDPsocket sock)

sock
停止し閉じて解放するUDPsocket

UDPsocketを停止し閉じて解放する. これを呼んだ後は, 新たに開かない限りUDPsocketを使ってはならない.

サンプルコード

// UDPsocketのチャネル0の全てのアドレスのバインドを解除する
//UDPsocket udpsock;

SDLNet_UDP_Close(udpsock);
udpsock=NULL; // これはこのUDPSocketが使えなくなったことを知る助けになる
