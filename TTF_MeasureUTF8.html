TTF_MeasureUTF8

与えられた幅に収まるUTF-8文字数を求める

構文

TTF_MeasureUTF8(TTF_Font *font, const char *text, int measure_width, int *extent, int *count)

引数

fontフォント
text求めるUTF-8テキスト
measure_width文字列の限界幅ピクセル数
extent文字列が収まる最小の幅ピクセル数
countレンダリング可能な文字数の代入先へのポインタ

戻り値

成功のとき0, エラーのとき-1を戻す.

詳細

measure_widthに達するまでレンダリングできる文字数を求める.

求めるために文字列をレンダリングする必要はない.

バージョン

SDL_ttf 2.0.18以降

関連項目(関数)

TTF_MeasureText
TTF_MeasureUNICODE
