Mix_GroupNewer

int Mix_GroupNewer(int tag)

tag
0以上のグループ番号. -1のとき全てのグループ

グループ番号tagで最も新しい再生中のチャネルを得る.

戻り値: 成功のときチャネル. グループに再生中のチャネルがない, またはグループが空のとき-1

サンプルコード

// グループ1で最も新しい再生中のチャネルを探す
int channel;
channel=Mix_GroupNewer(1);
if (channel==-1) {
    // 再生中または確保されたチャネルがない...
    // ここで単に利用可能なチャネルを探すとよいかもしれない...
}
