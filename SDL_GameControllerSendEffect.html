SDL_GameControllerSendEffect

コントローラー固有の効果パケットを送信する

構文

int SDL_GameControllerSendEffect(SDL_GameController* gamecontroller, const void* data, int size)

引数

gamecontrollerコントローラー
dataコントローラーに送信するデータ
sizeコントローラーに送信するデータの長さ

戻り値

成功のとき0, コントローラーまたはドライバが効果パケットに対応していないとき-1

バージョン

この関数はバージョン2.0.16以降で使える
