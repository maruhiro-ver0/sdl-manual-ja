Mix_FadingMusic

Mix_Fading Mix_FadingMusic()

音楽がフェードイン, アウト, フェードしていないかを得る. 音楽が再生中, 一時停止中かは得られない. 別にテストする必要がある.

戻り値: フェード状態. エラーを戻すことはない

サンプルコード

// 音楽のフェード状態をチェックする
switch(Mix_FadingMusic()) {
    case MIX_NO_FADING:
        printf("フェードしていない¥n");
        break;
    case MIX_FADING_OUT:
        printf("フェードアウト¥n");
        break;
    case MIX_FADING_IN:
        printf("フェードイン¥n");
        break;
}
