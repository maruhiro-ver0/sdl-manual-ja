TTF_CloseFont

void TTF_CloseFont(TTF_Font *font)

font
解放するTTF_Fontへのポインタ

fontが使用するメモリとfont自身をメモリから解放する. 新しいフォントを読み込まない限り, 解放後にfontを使ってはならない.

サンプルコード

// フォントを解放する
// TTF_Font *font;
TTF_CloseFont(font);
font=NULL; // 安全のため...
