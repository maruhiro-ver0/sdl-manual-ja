TTF_GetFontOutline

int TTF_GetFontOutline(TTF_Font *font)

font
アウトラインの幅を得るフォント

fontの現在のアウトラインの幅を得る.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: フォントに設定されたアウトラインの幅のピクセル数

サンプルコード

// 読み込まれたフォントのアウトライン幅を得る
//TTF_Font *font;
int outline=TTF_GetFontOutline(font);
printf("フォントのアウトライン幅は %d ピクセル¥n",outline);
