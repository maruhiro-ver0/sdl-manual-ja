IMG_LoadPNM_RW

SDL_Surface *IMG_LoadPNM_RW(SDL_RWops *src)

src
読み込み元のSDL_RWops. PNM画像はここから読み込まれる

PNM画像のsrcからをサーフェイスを生成する (SDL_imageライブラリがPNM形式に対応していれば).

戻り値: 新しいSDL_Surfaceへのポインタを戻す. PNMに対応していない, ファイルの読み込みに失敗したときなどはNULLを戻す.

サンプルコード

// sample.pnmをimageに読み込む
SDL_Surface *image;
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.pnm", "rb");
image=IMG_LoadPNM_RW(rwop);
if(!image) {
    printf("IMG_LoadPNM_RW: %s¥n", IMG_GetError());
    // ここでエラー処理を行う
}
