2.始めに

既にSDL_netを入手し, システムにインストールしていることを前提とする. SDL_netのソースコードアーカイブに含まれるINSTALL文書は, コンパイルとインストールの手助けになるだろう.

通常行うように, 以下の組み合わせでインストールする:

./configure
make
make install

いくつかのサンプルコードが次の場所でダウンロードできる:
http://www.jonatkins.org/SDL_net/
