Mix_GetNumChunkDecoders

int Mix_GetNumChunkDecoders()

Mix_GetChunkDecoderで利用可能なサンプルチャンクデコーダーの数を得る. それぞれの形式に対応した共有ライブラリの利用可能なチャンクデコーダーの数が変わるため, この数はプログラムの実行毎に異なる可能性がある.

戻り値: 利用可能なサンプルチャンクデコーダーの数

サンプルコード

// 利用可能なサンプルチャンクデコーダーの数を表示する
printf("%d つのサンプルチャンクデコーダーが利用可能である¥n", Mix_GetNumChunkDecoders());
