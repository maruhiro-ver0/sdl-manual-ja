Mix_FadeInMusic

int Mix_FadeInMusic(Mix_Music *music, int loops, int ms)

music
再生するMix_Music
loops
音楽を再生する回数. 0の場合は0回再生する... -1の場合は無限に再生する
ms
無音から最大ボリュームまでのフェードイン時間(ミリ秒)

読み込んだmusicを, msミリ秒でフェードインして, 最初から最後までloops回再生する. フェードインは最初のループのみ適用される. この関数を呼ぶ前に再生中の音楽が停止するかフェードアウトするまで待つ(ブロックされる). この関数はMix_FadeInMusicPos(music, loops, ms, 0)と同じである.

サンプルコード

// 音楽を2秒間でフェードインして無限に再生する.
// Mix_Music *music; // 読み込み済みとする
if(Mix_FadeInMusic(music, -1, 2000)==-1) {
    printf("Mix_FadeInMusic: %s¥n", Mix_GetError());
    // 音楽がない. しかし多くのゲームは音楽がなくても中断しない...
}

戻り値: 成功のとき0, 失敗のとき-1
