IMG_isGIF

int IMG_isGIF(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがGIF形式に対応しており, 画像データをGIFとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がGIFでSDL_ImageがGIF形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.gifがGIFかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.gif", "rb");
if(IMG_isGIF(rwop))
    printf("sample.gifはGIFファイルである¥n");
else
    printf("sample.gifはGIFファイルでないか, GIFに対応していない¥n");
