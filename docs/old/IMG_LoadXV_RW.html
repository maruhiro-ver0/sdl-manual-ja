IMG_LoadXV_RW

SDL_Surface *IMG_LoadXV_RW(SDL_RWops *src)

src
読み込み元のSDL_RWops. XV画像はここから読み込まれる

XV画像のsrcからをサーフェイスを生成する (SDL_imageライブラリがXV形式に対応していれば).

戻り値: 新しいSDL_Surfaceへのポインタを戻す. XVに対応していない, ファイルの読み込みに失敗したときなどはNULLを戻す.

サンプルコード

// sample.xvをimageに読み込む
SDL_Surface *image;
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.xv", "rb");
image=IMG_LoadXV_RW(rwop);
if(!image) {
    printf("IMG_LoadXV_RW: %s¥n", IMG_GetError());
    // ここでエラー処理を行う
}
