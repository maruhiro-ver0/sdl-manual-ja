IMG_LoadTGA_RW

SDL_Surface *IMG_LoadTGA_RW(SDL_RWops *src)

src
読み込み元のSDL_RWops. TGA画像はここから読み込まれる

TGA画像のsrcからをサーフェイスを生成する (SDL_imageライブラリがTGA形式に対応していれば). この関数でTGA画像以外を読み込もうとすると, TGA形式でなくても読み込みに成功してしまう. TGAにはマジックナンバーがなく, ファイルの形式を内容から識別する方法がないためである. よって慎重に扱わなければならない.

戻り値: 新しいSDL_Surfaceへのポインタを戻す. TGAに対応していない, ファイルの読み込みに失敗したときなどはNULLを戻す.

サンプルコード

// sample.tgaをimageに読み込む
SDL_Surface *image;
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.tga", "rb");
image=IMG_LoadTGA_RW(rwop);
if(!image) {
    printf("IMG_LoadTGA_RW: %s¥n", IMG_GetError());
    // ここでエラー処理を行う
}
