IMG_isPNM

int IMG_isPNM(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがPNM形式に対応しており, 画像データをPNMとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がPNMでSDL_ImageがPNM形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.pnmがPNMかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.pnm", "rb");
if(IMG_isPNM(rwop))
    printf("sample.pnmはPNMファイルである¥n");
else
    printf("sample.pnmはPNMファイルでないか, PNMに対応していない¥n");
