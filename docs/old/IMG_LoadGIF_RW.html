IMG_LoadGIF_RW

SDL_Surface *IMG_LoadGIF_RW(SDL_RWops *src)

src
読み込み元のSDL_RWops. GIF画像はここから読み込まれる

GIF画像のsrcからをサーフェイスを生成する (SDL_imageライブラリがGIF形式に対応していれば).

戻り値: 新しいSDL_Surfaceへのポインタを戻す. GIFに対応していない, ファイルの読み込みに失敗したときなどはNULLを戻す.

サンプルコード

// sample.gifをimageに読み込む
SDL_Surface *image;
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.gif", "rb");
image=IMG_LoadGIF_RW(rwop);
if(!image) {
    printf("IMG_LoadGIF_RW: %s¥n", IMG_GetError());
    // ここでエラー処理を行う
}
