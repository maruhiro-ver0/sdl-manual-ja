SDLNet_FreeSocketSet

void SDLNet_FreeSocketSet(SDLNet_SocketSet set)

set
メモリから解放するソケットセット

ソケットセットをメモリから解放する. 新たに確保する場合を除き, この関数を呼んだ後にsetを参照してはならない.

戻り値: なし. この呼び出しは常に成功する

サンプルコード

// ソケットセットを解放する
//SDLNet_SocketSet set;

SDLNet_FreeSocketSet(set);
set=NULL; // これはこのセットが確保されていないことを記憶する助けになる
