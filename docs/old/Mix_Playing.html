Mix_Playing

int Mix_Playing(int channel)

channel
再生中か否か調査するチャネル. -1のとき再生中のチャネル数を求める

channelが再生中か否かを調べる.

メモ: チャネルが一時停止中かはチェックしない.

戻り値: 再生中でなければ0, 引数に-1を渡したならば再生中のチャネル数, チャネルを指定して再生中ならば1を戻す.

サンプルコード

// サンプリング音声を再生中のチャネルの数をチェックする
printf("%d つのチャネルで再生中¥n", Mix_Playing(-1));
