TTF_FontFaceStyleName

char * TTF_FontFaceStyleName(const TTF_Font *font)

font
現在のスタイルの名前を得るフォント

fontの現在のスタイルの名前を得る. この関数は情報が得られなかったときNULLポインタを戻す.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: 指定のフォントの現在のスタイルの名前. NULLもありうる

サンプルコード

// フォントフェイスのスタイル名を得る
//TTF_Font *font;

char *stylename=TTF_FontFaceStyleName(font);
if(stylename)
    printf("フォントのスタイル名: %s¥n", stylename);
