IMG_isXV

int IMG_isXV(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがXV形式に対応しており, 画像データをXVとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がXVでSDL_ImageがXV形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.xvがXVPかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.xv", "rb");
if(IMG_isXV(rwop))
    printf("sample.xvはXVファイルである¥n");
else
    printf("sample.xvはXVファイルでないか, XVに対応していない¥n");
