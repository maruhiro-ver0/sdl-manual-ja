SDLNet_UDP_Unbind

void SDLNet_UDP_Unbind(UDPsocket sock, int channel)

sock
アドレスのバインドを解除するUDPsocket
channel
UDPsocketのアドレスのバインドを解除するチャネル

ソケットのチャネルに割り当てた(バインドした)全てのアドレスを解除する. その後に新しいアドレスをソケットのチャネルにバインドできるようになる.

戻り値: なし. 常に成功する

サンプルコード

// UDPsocketのチャネル0の全てのアドレスのバインドを解除する
//UDPsocket udpsock;

SDLNet_UDP_Unbind(udpsock, 0);

SDLNet_UDP_Bind, SDLNet_UDP_Close, UDPsocket
