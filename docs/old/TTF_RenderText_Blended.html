TTF_RenderText_Blended

SDL_Surface *TTF_RenderText_Blended(TTF_Font *font, const char *text, SDL_Color fg)

font
テキストをレンダリングするフォント
text
レンダリングするNUL終端のLATIN1文字列
fg
レンダリングするテキストの色. ピクセルは透明色とこの色でブレンドされ, アンチエイリアスされたグリフが描かれる

LATIN1でエンコードされたtextを, fontと色fgで新しく生成したサーフェイスに混合(Blend)モードでレンダリングする. 呼び出し側(あなたのことだ!)が戻されたサーフェイスを解放しなければならない.

メモ: この関数のfontにNULLを渡すと異常終了する.

メモ: この関数のtextにNULLを渡した場合の動作は未定義である.

戻り値: 生成されたSDL_Surface. 失敗のときNULL

サンプルコード

// 黒色で新しいサーフェイスにテキストをレンダリングする
// そしてそれを画面の左上にコピーする
// そしてテキストのサーフェイスを解放する
//SDL_Surface *screen;
SDL_Color color={0,0,0};
SDL_Surface *text_surface;
if(!(text_surface=TTF_RenderText_Blended(font,"Hello World!",color))) {
    // ここでエラーを処理する. 少なくともTTF_GetErrorを表示したほうがよい
} else {
    SDL_BlitSurface(text_surface,NULL,screen,NULL);
    // これは再利用できる. しかしここでは単純化のためにそうしない
    SDL_FreeSurface(text_surface);
}
