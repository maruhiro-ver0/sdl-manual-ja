TTF_FontFaces

long TTF_FontFaces(const TTF_Font *font)

font
利用可能なフェイスの数を得るフォント

fontの利用可能なフェイス(サブフォント)の数を得る. これは自分自身を含むフォント(サイズ, スタイル, その他の字体の属性に基づく)の数である. この数は他のSDL_ttf関数では使えないので, わかっても無意味と思われる.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: 指定のフォントのフェイスの数

サンプルコード

// 読み込まれたフォントのフェイスの数を得る
//TTF_Font *font;

printf("フォントのフェイス数: %ld¥n", TTF_FontFaces(font));
