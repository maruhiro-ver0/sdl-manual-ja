IMG_isXCF

int IMG_isXCF(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがXCF形式に対応しており, 画像データをXCFとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がXCFでSDL_ImageがXCF形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.xcfがXCFかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.xcf", "rb");
if(IMG_isXCF(rwop))
    printf("sample.xcfはXCFファイルである¥n");
else
    printf("sample.xcfはXCFファイルでないか, XCFに対応していない¥n");
