IMG_isPNG

int IMG_isPNG(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがPNG形式に対応しており, 画像データをPNGとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がPNGでSDL_ImageがPNG形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.pngがPNGかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.png", "rb");
if(IMG_isPNG(rwop))
    printf("sample.pngはPNGファイルである¥n");
else
    printf("sample.pngはPNGファイルでないか, PNGに対応していない¥n");
