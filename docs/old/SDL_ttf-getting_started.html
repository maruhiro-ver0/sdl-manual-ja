2.始めに

既にSDL_ttfを入手し, システムにインストールしていることを前提とする. SDL_ttfのソースコードアーカイブに含まれるINSTALL文書は, コンパイルとインストールの手助けになるだろう.

通常行うように, 以下の組み合わせでインストールする:

./configure
make
make install

SDL_ttfはTrueTypeファイルに対応している. ファイル名は通常は .ttf で終わるが, 一部 .fon のファイルもある. ほとんどのフォントには著作権がある. 配布する前にフォントのライセンスを確認すること.

いくつかのフリーのフォント:

  • Free UCS Outline Fonts
  • Fonthead Design
  • Bitstream Vera Fonts
  • FreeUniFont
  • 1001 Fonts
  • Google!

(訳者注: 日本語フォントならば, VLゴシックフォント, 花園フォント(花園明朝), IPAフォント, M+ FONTS等がある.)

サンプルコードが必要ならば, 次の場所でダウンロードできる:
