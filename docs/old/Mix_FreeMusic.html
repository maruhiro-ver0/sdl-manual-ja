Mix_FreeMusic

void Mix_FreeMusic(Mix_Music *music)

music
解放するMix_Music

読み込んだmusicを解放する. musicの再生は停止する. musicがフェードアウト中ならば, この関数は完了するまで待つ(ブロックされる).

サンプルコード

// 音楽を解放する
Mix_Music *music;
Mix_FreeMusic(music);
music=NULL; // 解放したとわかるようにするため...
