IMG_Quit

void IMG_Quit()

この関数は読み込んだ動的ライブラリをメモリから解放する. もし再びライブラリが必要な場合は, IMG_Init関数または画像の読み込みで動的ライブラリを読み込む必要がある. この関数はIMG_LoadでJPG, PNG, TIF画像を読み込む必要がなくなったとき呼ぶとよい. 何度IMG_Init関数を呼んだとしても, この関数は一度だけ呼べばよい.

サンプルコード

// 動的に読み込まれた画像ライブラリを解放する
// 
IMG_Quit();
