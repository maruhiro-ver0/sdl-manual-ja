TTF_GetFontKerning

int TTF_GetFontKerning(TTF_Font *font)

font
カーニングを得るフォント

fontの現在のカーニング(文字詰め)の有無を得る.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: カーニングが無効のとき0, 有効のとき0以外. フォントを読み込んだ直後は1(有効)である.

サンプルコード

// 読み込まれたフォントのカーニングを得る
//TTF_Font *font;
int kerning=TTF_GetFontKerning(font);
printf("フォントの現在のカーニングは %s¥n",
        kerning==0?"無効":"有効");
