Mix_PlayingMusic

int Mix_PlayingMusic()

音楽を再生しているか否かを得る.

メモ: 一時停止中かはチェックしない.

戻り値: 再生中でない場合は0, 再生中の場合は1

サンプルコード

// 音楽が再生中かチェックする
printf("音楽は再生中%s¥n", Mix_PlayingMusic()?"である":"でない");
