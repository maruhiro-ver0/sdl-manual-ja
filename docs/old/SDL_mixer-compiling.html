2.2 コンパイル

SDL_mixerをリンクするためには, sdl-configでコンパイルオプションを獲得する必要がある. それ以外は, SDL_mixerを使ってコンパイルすることは簡単である.
注意: 一部のシステムでは, SDL_mixerのライブラリとインクルードファイルは, SDLのライブラリとインクルードファイルと同じ場所にない. その場合は, さらに -I と -L でパスを加える必要がある.

コンパイルの例:
cc -c `sdl-config --cflags` mysource.c

リンクの例:
cc -o myprogram mysource.o `sdl-config --libs` -lSDL_mixer

これでプログラムを実行できるようになる.
