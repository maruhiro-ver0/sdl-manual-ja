TTF_Init

int TTF_Init()

TrueTypeフォントAPIを初期化する. このライブラリのTTF_WasInitを除く関数を使う前にこの関数を呼ぶ必要がある. この関数の前にSDLが初期化されている必要はない.

戻り値: 成功のとき0, 失敗のとき-1

サンプルコード

if(TTF_Init()==-1) {
    printf("TTF_Init: %s¥n", TTF_GetError());
    exit(2);
}
