IMG_isTIF

int IMG_isTIF(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがTIF形式に対応しており, 画像データをTIFとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がTIFでSDL_ImageがTIF形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.tifがTIFかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.tif", "rb");
if(IMG_isTIF(rwop))
    printf("sample.tifはTIFファイルである¥n");
else
    printf("sample.tifはTIFファイルでないか, TIFに対応していない¥n");
