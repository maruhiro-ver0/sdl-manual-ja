2.2 コンパイル

SDL_netをリンクするためには, sdl-configでコンパイルオプションを獲得する必要がある. それ以外は, SDL_netを使ってコンパイルすることは簡単である.
注意: 一部のシステムでは, SDL_netのライブラリとインクルードファイルは, SDLのライブラリとインクルードファイルと同じ場所にない. その場合は, さらに -I と -L でパスを加える必要がある. 例はgccとUNIXを想定しているが, 他のコンパイラとOSにも適用できる.

コンパイルの例:
gcc -c `sdl-config --cflags` mysource.c

リンクの例:
gcc -o myprogram mysource.o `sdl-config --libs` -lSDL_net

これでプログラムを実行できるようになる.
