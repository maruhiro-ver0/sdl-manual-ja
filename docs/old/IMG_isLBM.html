IMG_isLBM

int IMG_isLBM(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがLBM形式に対応しており, 画像データをLBMとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がLBMでSDL_ImageがLBM形式に対応していれば1, そうでなければ0.

サンプルコード

// sample.lbmがLBMかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.lbm", "rb");
if(IMG_isLBM(rwop))
    printf("sample.lbmはLBMファイルである¥n");
else
    printf("sample.lbmはLBMファイルでないか, LBMに対応していない¥n");
