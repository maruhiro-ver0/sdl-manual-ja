Mix_FreeChunk

void Mix_FreeChunk(Mix_Chunk *chunk)

chunk
解放するMix_Chunk

chunkが使うメモリとchunk自身を解放する. この関数の後に新しいサンプリング音声を読み込まずにchunkを使ってはならない.

メモ: 再生中にchunkを解放するべきではない...

サンプルコード

// サンプリング音声を解放する
// Mix_Chunk *sample;
Mix_FreeChunk(sample);
sample=NULL; // 安全のため...
