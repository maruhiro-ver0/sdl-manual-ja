IMG_isICO

int IMG_isICO(SDL_RWops *src)

src
テストする画像のSDL_RWops

ライブラリがICO形式に対応しており, 画像データをICOとして読み込めるかテストする. 読み込めないとき偽(0)を戻す.

戻り値: 画像がICOでSDL_ImageがICO形式とBMP形式に対応していれば1, そうでなければ0

サンプルコード

// sample.icoがICOかテストする
SDL_RWops *rwop;
rwop=SDL_RWFromFile("sample.ico", "rb");
if(IMG_isICO(rwop))
    printf("sample.icoはICOファイルである¥n");
else
    printf("sample.icoはICOファイルでないか, BMPに対応していない¥n");
