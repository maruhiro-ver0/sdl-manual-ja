Mix_QuickLoad_RAW

Mix_Chunk *Mix_QuickLoad_RAW(Uint8 *mem)

mem
WAVEファイルが書き込まれたメモリバッファ

memの生のサンプリング音声データを読み込む. データは出力形式と同じである必要がある. 自分が何をしているのか理解していないならば, この関数を使うべきではない!

メモ: この関数は最小限のチェックしかしない. ファイルの形式と出力形式が異なる, またはWAVEファイルでない場合でもエラーを戻さない. この関数を使うのは危険な場合がある.

戻り値: サンプリング音声のMix_Chunkへのポインタ. 失敗のときNULL

サンプルコード

// メモリから生のサンプリング音声を高速に読み込む
// Uint8 *raw; // 生データが書き込まれている
               // またはプログラム内で書き込むとする...
Mix_Chunk *raw_chunk;
if(!(raw_chunk=Mix_QuickLoad_RAW(raw))) {
    printf("Mix_QuickLoad_RAW: %s¥n", Mix_GetError());
    // ここでエラーを処理する
}
