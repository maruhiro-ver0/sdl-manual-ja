Mix_MusicType

typedef enum {
    MUS_NONE,
    MUS_CMD,
    MUS_WAV,
    MUS_MOD,
    MUS_MID,
    MUS_OGG,
    MUS_MP3, /* SMPEGを使用 */
    MUS_MP3_MAD,
    MUS_FLAC
} Mix_MusicType;

Mix_GetMusicTypeの戻り値はこの列挙体の値である. 音楽が再生中でなければMUS_NONEが戻される. 外部プレイヤーで再生中ならばMUS_CMDが戻される. その他は名前の通りである.
