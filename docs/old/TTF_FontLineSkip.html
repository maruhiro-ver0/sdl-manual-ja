TTF_FontLineSkip

int TTF_FontLineSkip(const TTF_Font *font)

font
行の高さを得るフォント

fontの行の推奨高さピクセル数を得る. これは通常はTTF_FontHeightより大きい.

メモ: この関数のfontにNULLを渡すと異常終了する.

戻り値: 指定のフォントの全てのグリフの中で最大の高さのピクセル数

サンプルコード

// 読み込まれたフォントの行の高さを得る
//TTF_Font *font;

printf("フォントの行の高さ: %d¥n", TTF_FontLineSkip(font));
