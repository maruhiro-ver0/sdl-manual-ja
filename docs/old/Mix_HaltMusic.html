Mix_HaltMusic

int Mix_HaltMusic()

(フェーディング中でも)音楽の再生を停止する. 音楽が停止したときMix_HookMusicFinishedで設定したコールバック関数が呼ばれる.

戻り値: 常に0を戻す. (Mix_HaltGroupよりも馬鹿げている)

サンプルコード

// 音楽の再生を停止する
Mix_HaltMusic();

Mix_FadeOutMusic, Mix_HookMusicFinished
