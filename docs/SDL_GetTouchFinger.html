SDL_GetTouchFinger

指定のタッチデバイスIDの指番号の指オブジェクトを得る

構文

SDL_Finger* SDL_GetTouchFinger(SDL_TouchID touchID, int index)

引数

touchIDタッチデバイスのID
index指の番号

戻り値

SDL_Fingerオブジェクトの戻す. 指定のIDと指番号がない場合はNULLを戻す.

詳細

戻されたオブジェクトはSDLの管理下あるので解放してはならない.

バージョン

SDL 2.0.0以降
