SDL_size_add_overflow

aとbの和を求めオーバーフローしたとき-1を戻す

構文

int SDL_size_add_overflow (size_t a, size_t b, size_t *ret)

引数

a値a
b値b
ret和へのポインタ

戻り値

オーバーフローしたとき-1を戻す. オーバーフローしなかったときretに和を設定し0を戻す.

バージョン

SDL 2.24.0以降
