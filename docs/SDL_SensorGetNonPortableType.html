SDL_SensorGetNonPortableType

センサーの環境に依存した種類を得る

構文

int SDL_SensorGetNonPortableType(SDL_Sensor *sensor)

引数

sensor調査するSDL_Sensorオブジェクト

戻り値

環境に依存したセンサーの種類. sensorがNULLのときは-1を戻す.

バージョン

SDL 2.0.9以降
