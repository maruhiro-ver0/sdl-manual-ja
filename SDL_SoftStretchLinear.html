SDL_SoftStretchLinear

2つのサーフェイス(32ビット形式)の間で双一次補間(biliner)で拡大縮小してコピーする YUV変換モードを設定する

構文

int SDL_SoftStretchLinear(SDL_Surface* src, const SDL_Rect* srcrect, SDL_Surface* dst, const SDL_Rect* dstrect)

バージョン

SDL 2.0.16以降
